# 下载

转到 [Release 页面](https://github.com/Sashwind888/MapTP-translate/releases/latest) 以下载最新构建版本

## 使用步骤
**注意**: MapTP 需要 *精确式触摸板设备 (PTP)* 以正常运行。
* 下载并解压二进制文件。
* 打开 `MapTP_ZH.exe`。
* 点击 `校准触摸板尺寸`。
* 点击触摸板的右下角。 (把手指滑动一下，以确保窗口中的数字达到最大值)
* 按下 好的(Enter) 按钮 
* 点击 `设置为屏幕大小`

开始享受!

## 关于 "海龟模式"

参见 [#3](https://github.com/lingrottin/MapTP/issues/3): 在开启海龟模式后，MapTP 会在你按下手指的同时按下鼠标左键，移开手指的同时释放，如同触屏设备(数位板?)的工作逻辑。也许对绘图有帮助。
